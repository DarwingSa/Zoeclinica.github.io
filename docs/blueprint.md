# **App Name**: VetPet Haven

## Core Features:

- Informative Home Section: Display welcome text, call to action, and a relevant image using Bootstrap's grid system.
- Service Showcase: Use Bootstrap cards and grid to showcase services like consultations and surgeries, and the 'Tramites de Viajes Internacionales'.
- Advanced Diagnostics Explanation: Detail the services available through Clinical Laboratory and Digital X-rays.
- Contact Information and Form: Provide contact information, a Google Maps embed, and a Bootstrap form for inquiries.
- International Travel Guidance Tool: AI tool uses LLM to consider information such as the user's destination, animal type, animal age and known health conditions, to respond with appropriate procedures for traveling internationally with a dog or cat. Response may contain details regarding health certificates, microchipping, and required advisories. Should only respond for travel from country in which clinic operates.

## Style Guidelines:

- Background color: White (#FFFFFF) for a clean and professional look.
- Primary color: Light blue (#ADD8E6) to evoke trust and calmness.
- Accent color: Darker blue (#3B5998) for interactive elements and important text, providing contrast. 
- Body and headline font: 'Inter', a sans-serif font for a modern and clean feel.
- Utilize Bootstrap's grid system to ensure a responsive layout across devices.
- Incorporate a fixed navbar at the top for easy navigation to different sections.
- Use line icons to represent different services, maintaining a clean and minimal design.
